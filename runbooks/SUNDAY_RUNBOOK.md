# Sunday Runbook — Mezei Meal Prep

## Checklist

### A) Intake (before planning)
- [ ] Inventory updated in JSON (fridge/pantry/freezer)
- [ ] Perishable estimates set where known
- [ ] This week’s dislikes/constraints set
- [ ] Calorie/protein targets confirmed
- [ ] Prep time budget confirmed

### B) Generate
- [ ] Open web app and tap generate (or run planner script)
- [ ] Review generated plan + validation
- [ ] Confirm PASS (or acceptable WARN only)
- [ ] Download/save weekly-plan + validation outputs

### C) Adjust
- [ ] Remove meals that feel unrealistic
- [ ] Confirm shrimp frequency acceptable
- [ ] Confirm prep complexity fits available time
- [ ] Re-run after edits if needed

### D) Execute Prep
- [ ] Batch cook staples (grains/beans)
- [ ] Prep vegetables + sauces
- [ ] Cook main proteins (tofu/tempeh/shrimp)
- [ ] Portion 3-4 days, freeze extras

### E) Midweek + Closeout
- [ ] Wednesday check: update use-up priorities
- [ ] Friday note: liked/disliked meals, leftovers/waste
- [ ] Carry those notes into next Sunday intake
