# Validation Heuristics

Use these checks every weekly run before prep.

## Diet Safety
- Vegetarian-compliant (no meat/fish except shrimp)
- Shrimp frequency capped by preference (default <=3 slots/week)

## Weight-Loss Alignment
- Avg daily calories near target (+/- 200 preferred, +/- 250 max)
- Protein not materially below target (prefer within 15g)
- At least one high-protein option available daily

## Practicality
- Prep plan fits available Sunday minutes
- Shopping list grouped and minimal
- Uses inventory first (especially near-expiry items)

## Variety & Adherence
- >=4 unique core meals/week
- No disliked ingredient in core meal names
- Includes fallback swaps for busy/stockout days

## Waste Reduction
- Explicit use-up-first list present
- Midweek adjustment note present
