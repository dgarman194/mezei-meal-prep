# Meal Library Direction (Practical, Not Overbuilt)

Goal: expand weekly variety while keeping prep simple and repeatable.

## Library shape
Keep a small, high-performing core:
- **6 vegetarian batch mains**
- **2 shrimp quick mains**
- **4 protein booster snacks**

Do not add dozens of recipes at once.

## Add meals by "template families"
Each family should share prep flow + shopping overlap.

1. **Bowl Family** (grain + protein + veg + sauce)
2. **Salad Box Family** (protein + crunchy veg + dressing)
3. **Skillet Family** (quick stir-fry/sauté)
4. **Soup/Stew Family** (freezer-friendly comfort)

## Flavor rotation by week
Use one primary profile + one secondary profile:
- Mediterranean + Tex-Mex
- Curry + Lemon-Herb
- Soy-Ginger + Mediterranean

This creates variety without exploding ingredient count.

## Acceptance test for any new meal
A new meal enters the planner only if:
- prep fits Sunday batch workflow
- reheats well or is intentionally cold/fresh
- uses overlapping ingredients with existing meals
- has a clear taste-upgrade instruction
- supports calorie/protein targets
