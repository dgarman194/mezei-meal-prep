# Weekly Operator Packet

Use this packet every Sunday when running the planning assistant.

## Inputs to provide
1. Intake JSON (inventory + preferences + targets)
2. Any special week constraints (events, travel, low-time days)

## Instruction wrapper
"Generate this week's meal-prep plan using the required output sections.
Favor ingredients already on hand, prioritize perishables, and keep prep realistic.
Do not include non-vegetarian proteins except shrimp when enabled."

## Manual review checklist
- Diet compliant
- Prep feasible
- Shopping list minimal
- Macro estimates near target
- At least one quick rescue meal
